# Eshwar Chavan — Portfolio

Personal portfolio built with Next.js 15, React 19, TypeScript, Tailwind CSS, and Framer Motion.
Content is sourced directly from resume and GitHub (https://github.com/eshwarchavan05) — no fabricated projects, metrics, or experience.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Build

```bash
npm run build
npm start
```
